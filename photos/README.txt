WAREHOUSE PHOTOS — NAVCOR USDA AMS Presentation
================================================

SINGLE PHOTO (simplest):
  Save as NAME.jpg or NAME.png. Just one photo per location.

MULTIPLE PHOTOS (up to 6 per location, gallery view):
  Save as NAME-1.jpg, NAME-2.jpg, NAME-3.jpg, ... up to NAME-6.jpg
  The gallery will automatically:
    - Show nav dots at the bottom of the banner
    - Show ← / → arrows on hover
    - Show a "2 / 4" counter at the top-left
    - Let USDA scroll/swipe horizontally between photos

BACKWARD COMPATIBLE:
  If you have NAME.jpg already AND want to add more, you don't need to rename.
  Just add NAME-2.jpg, NAME-3.jpg, NAME-4.jpg etc.
  Slot 1 prefers NAME-1.jpg, falls back to NAME.jpg if -1 isn't found.

FILENAMES BY LOCATION:
  smithfield-1.jpg, smithfield-2.jpg, smithfield-3.jpg, smithfield-4.jpg
  houston-1.jpg, houston-2.jpg, ...
  jacksonville-1.jpg, jacksonville-2.jpg, ...
  jersey-1.jpg, jersey-2.jpg, ...

EXAMPLE — YOU HAVE 4 PHOTOS FOR SMITHFIELD:
  Save them as:
    smithfield-1.jpg   ← appears first
    smithfield-2.jpg
    smithfield-3.jpg
    smithfield-4.jpg
  Refresh the HTML. The card automatically shows: dots (●○○○), arrows on hover,
  counter "1 / 4" top-left, and USDA can click/swipe through all four.

NOTES:
  - JPG preferred (smaller file size). PNG also works.
  - Wide horizontal photos look best (16:9 or 3:2).
  - Minimum 1200 px wide recommended for crisp display.
  - Both .jpg and .png are tried automatically — use whichever you have.
